# AdaptiveCards-Mobile
Adaptive Cards Mobile: Next-generation Adaptive Cards SDK for iOS (SwiftUI) and Android (Jetpack Compose), designed for Microsoft Teams mobile integration.

## Platform Status

- ✅ **iOS SDK**: Complete - See [ios/README.md](ios/README.md)
- ⏳ **Android SDK**: Pending implementation

## Cross-Platform Consistency

This project maintains naming and design convention alignment across iOS and Android platforms. See [CROSS_PLATFORM_ALIGNMENT.md](CROSS_PLATFORM_ALIGNMENT.md) for details on how the platforms should align once both are implemented.
